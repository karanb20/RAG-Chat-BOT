# 🤖 RAG Chatbot (PDF-based)

An advanced Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDFs and ask questions based on the document content.

---

## 🚀 Features

- 📂 Upload any PDF
- 💬 ChatGPT-like conversational UI
- 🧠 Context-aware responses using memory
- 🔍 Semantic search using FAISS
- 📚 Source document references
- ⚡ Fast responses using Groq LLM

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- FAISS (Vector Database)
- HuggingFace Embeddings
- Groq API (LLM)
